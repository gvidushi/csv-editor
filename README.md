# CSV Editor 

Lightweight CSV editor Svelte component built on top of Statue site.

# Features incorporated
- Load CSV from disk
- Display CSV in table
- Edit cells
- Add rows
- Deltee rows
- Download (modified) CSV
- Filter rows (broken)

# Software
- Svelte and SvelteKit (statue starter code)
- PapaParse for CSV parsing
